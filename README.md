# 🔢 Jogo Secreto dos Números

Este projeto é um jogo de adivinhação interativo onde o usuário tenta descobrir um número secreto gerado pelo sistema.Ele representa minha transição do desenvolvimento estático com HTML e CSS para a criação de aplicações dinâmicas utilizando JavaScript.

## 🛠️ Tecnologias e Conceitos Aplicados

- **JavaScript (ES6+)**: Utilizado para toda a lógica do jogo  
- **Manipulação do DOM**: Integração do script com elementos HTML (texto, botões e inputs)  
- **Voz responsiva**: Implementação de síntese de voz para tornar o jogo mais acessível e interativo  
- **HTML5 e CSS3**: Estrutura e estilização da interface

## 📖 Jornada de Desenvolvimento (Lógica de Programação)

O projeto foi desenvolvido com base em dois módulos fundamentais da Alura:

### 🔹 1. Introdução à programação com JavaScript

- **Carga horária**: 6 horas  
- **Curso**: https://cursos.alura.com.br/course/logica-programacao-mergulhe-programacao-javascript  

**Desafio:**  
Construir a base do jogo onde o sistema gera um número aleatório e o usuário precisa adivinhar.

**Solução e Aprendizados:**  
- Uso de variáveis  
- Estruturas condicionais (`if/else`)  
- Laços de repetição (`while`)  
- Primeira experiência prática com lógica e algoritmos  

---

### 🔹 2. Explorando Funções e Listas
**Desafio:**  
Refatorar o código para torná-lo mais organizado e melhorar a experiência do usuário.

**Solução e Aprendizados:**  
- **Funções**: Organização do código com parâmetros e retornos  
- **Arrays (listas)**: Evitar repetição de números sorteados  
- **Interatividade**: Manipulação do DOM para mensagens dinâmicas  
- Limpeza automática do campo de entrada

### 🎮 Instruções

1. Digite um número no campo de entrada  
2. Clique em **"Adivinhar"**  
3. O sistema informará se o número secreto é maior ou menor  
4. Tente acertar com o menor número de tentativas possível! 🎯
